Smooth theme for the imageLightbox plugin

== Description ==

Theme for the imageLightbox plugin: http://wordpress.org/plugins/imagelightbox/

== Installation ==
1. Install and activate the imageLightbox plugin (http://wordpress.org/plugins/imagelightbox/)
2. Install and activate this plugin

== Frequently Asked Questions ==

= Do I need to install the imageLightbox plugin =
Yes

= Does this theme work with with other imageLightbox plugins =
No

= Does this theme work with imageLightbox installed with my theme =
No

= Does this theme work with with other lightbox plugins =
No

= Why is this theme released as a plugin and not as a theme =
Because it is essentially an extension to the imageLightbox plugin, not a WordPress theme.
